Concept of WWW

  • It stands for world wide web
  • It is basically a system of internet servers that support specially formatted documents
  • It is a collection of text pages,videos,digital photograps and animations we can access over the internet
  • User acces the world wide web facilities via client called web browser.

HTTP Protocol

  1. HTTP stands for hypertext transfer protocol
  2. HTTP is based on the client server architecture model and a stateless request/responce protocol
  3. 404 File Not Found is a common HTTP status code.
